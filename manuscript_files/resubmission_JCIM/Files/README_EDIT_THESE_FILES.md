# Editable revision sources

This is the only directory containing editable LaTeX sources for the revision.

Do not edit files in `../build/`: the clean and marked outputs are regenerated from the
sources here by running `make` in `resubmission_JCIM/`.

`audit_materials/` contains the separate SEM/autocorrelation audit note and its data; it
is not included in the manuscript or Supporting Information build.
