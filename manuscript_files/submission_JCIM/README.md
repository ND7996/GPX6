# JCIM submission package — 11 May 2026

This directory reconstructs the submitted 11 May 2026 version from Git commit `7d98be3`. The manuscript and Supporting Information PDFs match the archived PDFs byte-for-byte.

- `Files/`: LaTeX sources, bibliography and compilation tools.
- `Figures/`: independent figure set used by this version.
- `Cover_letter_JCIM.*`: final cover letter source files.
- `GPX6_manuscript_JCIM.pdf` and `GPX6_suppinfo_JCIM.pdf`: archived submitted outputs.

Run `make` in this directory to rebuild the manuscript and SI. Rebuilt outputs use the `_rebuilt.pdf` suffix so the exact archived delivery files are never overwritten.
